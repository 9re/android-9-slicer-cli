====

## android-9-slicer-cli

====

## installation

```sh
$ cmake .
$ make
$ sudo make install
```

====

## usage

```sh
$ 9slice -o <output> [-d <dpi of input> |
                      -i <inset_top> <inset_left> <inset_bottom> <inset right> |
                      -p <padding_top> <padding_left> <padding_bottom> <padding_right>]
                      <input>
```
